# leiriense_scrapper

A simple Python script project scaffolded by Copilot.

## Usage

```bash
python main.py
```

### Category extraction

The `category_extractor.py` script demonstrates using Selenium to
download a fully rendered page before parsing it with BeautifulSoup:

```bash
pip install selenium webdriver-manager beautifulsoup4
python category_extractor.py
```

## Description

This repository will eventually contain scrapers for the Leiriense website.
